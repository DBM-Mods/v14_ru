# Установка модов
Возьмите все файлы из ZIP и замените в папку вашего проекта<br>
Поместите файл " Русский.json "в папке вашего DBM" диск: \ Steam\steamapps \ common\Discord Bot Maker\translations " <br>

# NODE JS требуется

Версия 22.14.0
